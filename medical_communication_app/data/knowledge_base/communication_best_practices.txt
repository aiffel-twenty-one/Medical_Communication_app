# 의료진 소통 모범 사례

1. **경청하기**: 환자의 말을 끊지 않고 끝까지 경청한다.

2. **명확한 표현**: 의료 용어를 사용하기보다 쉽게 이해할 수 있는 언어로 설명한다.

3. **공감 표현**: 환자의 감정과 상태에 공감하고 이해한다는 표현을 한다.

4. **질문 유도**: 환자가 궁금한 점을 자유롭게 질문할 수 있도록 분위기를 조성한다.

5. **요약 제공**: 중요한 내용을 다시 한 번 요약하여 환자가 이해했는지 확인한다.

# 의사 소통 개선 방법

- 시각적 자료 활용: 필요한 경우 그림이나 도표를 활용하여 설명한다.
- 피드백 요청: 환자에게 이해 여부를 물어보고 추가 설명이 필요한 부분을 파악한다.
- 문화적 배경 고려: 환자의 문화적, 사회적 배경을 고려하여 소통한다.
